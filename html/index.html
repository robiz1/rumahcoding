seharah html dan css

apa itu html

  • HTML singkatan dari Hyper Text Markup Language dan digunakan untuk menulis halaman web.
  • HTML adalah bahasa Markup
  • Sebuah file HTML adalah file teks yang berisi Markup Tag.
  • The Markup Tag menginstruksikan browser Web bagaimana menampilkan informasi halaman.
  • Dokumen HTML dapat berisi teks, gambar, link , dan konten multimedia tambahan dan skrip pemrograman.
  • Untuk membuat sebuah file HTML editor teks sederhana ( notepad ) atau beberapa editor HTML modern ( Dreamweaver ) dapat digunakan.
  • Sebuah file HTML harus memiliki .htm atau .html sebagai ekstensi.
